\subsection*{Double interpretation of Bayesian methods: Average vs learning}

Looking at the product for a fixed $\delta$ and then integrating (viewed as averaging over unknown parameters).

Integrating over the posterior for each block and then taking the product (viewed as learning).

Let $S(X_i \mid \delta)$ be the SEQ-COND E-variable that we consider.
Assume that we do not know the oracle alternative $\delta^*$ for now but there is some ways to \emph{learn} or \emph{average} them.

Recall the factorization of joint distribution of $X^n=\{X_1,X_2,\dots,X_n\}$:
\begin{align*}
	P(X_1,X_2,\dots,X_n) = P(X_1) P(X_2\mid X_1) P(X_3\mid X_1,X_2) \cdots P(X_n\mid X^{n-1})
	= \prod_{i=1}^n P(X_i \mid X^{i-1}).
\end{align*}

Note that this does not require \emph{iid}!

For iid $X_i$:
\begin{align*}
	P(X^n) &= \prod_{i=1}^n P(X_i) = \int \pi(\delta) \prod_{i=1}^n P(X_i) \dd{\delta} \qq{Aveging} \\
	&= \prod_{i=1}^n \int P(X_i \mid \delta) \pi(\delta \mid X^{i-1}) \dd{\delta} \qq{Learning}
\end{align*}

Note that it always holds that
\begin{align*}
	S(X^n) = \int S(X^n\mid\delta)\pi(\delta) \dd{\delta} = \prod_{i=1}^n S(X\mid \delta)
\end{align*}

\subsection*{Prequential Plug-in NML}

Assume parametric model and simple null and alternative hypothesis.
We observe iid data stream $X^t=\{X_1,X_2,\dots,X_t\}$.
To make it even simpler, the support of $X$: $\Xc$ is finite.

\begin{align*}
	\Hc_0 &: X \sim P(X \mid \theta_0) \\
	\Hc_1 &: X \sim P(X \mid \theta^*)
\end{align*}

Now let's look at the plug-in, for the first observation $X_1$.
\begin{equation*}
	S_{\mathrm{NML}}^1 = \frac{P(X_1 \mid \hat{\theta}(X_1))}{P(X_1 \mid \theta_0) \sum_{k\in\Xc} P(k \mid \hat{\theta}(k))},
\end{equation*}
where $\hat{\theta}(\cdot)$ is the MLE estimator.

Check this is a E-variable:
\begin{equation*}
	\ERWi{0}{S^1_{\mathrm{NML}}} = 
	\sum_{X\in\Xc} P(X\mid \theta_0) \cdot 
		\frac{P(X\mid \hat{\theta}(X))}{P(X\mid\theta_0) \sum_{k\in\Xc} P(k \mid \hat{\theta}(k))}
	= \sum_{X\in\Xc}\frac{P(X\mid\hat\theta(X))}{\sum_{k\in\Xc}P(k\mid\hat{\theta}(k))}
	\leq 1.
\end{equation*}

For the second time point, we shift the $X_2$ over $\Xc$ and obtained a MLE estimator:
\begin{equation*}
	S_{\mathrm{NML}}^2 = \frac{P(X_2 \mid \hat{\theta}(X_1,X_2))}{
		P(X_2 \mid \theta_0) \sum_{k\in\Xc} P(k \mid \hat{\theta}(X_1, k))
	}
\end{equation*}

Going to time $t$
\begin{equation*}
	S_{\mathrm{NML}}^t = \frac{P(X_t \mid \hat{\theta}(X_1,X_2,\dots,X_t))}{
		P(X_t \mid \theta_0) \sum_{k\in\Xc} P(k \mid \hat{\theta}(X_1,X_2,\dots, k))
	}
	= \frac{P(X_t \mid \hat{\theta}(X^t))} {
		P(X_t \mid \theta_0) \sum_{k\in\Xc} P(k \mid \hat{\theta}(\tilde{X}^t))
	}
\end{equation*}

Given the canonical filtration $\Fc_{t-1}$,
\begin{equation*}
	\condERWi{0}{S^t_{\mathrm{NML}}}{\Fc_{t-1}}
	= \sum_{\Xc} \frac{P(X_t\mid \hat{\theta}(X_1,X_2,\dots,X_{t-1},X_t))}{
	\sum_{k\in\Xc}P(k \mid \hat{\theta}(X_1,X_2,\dots,X_{t-1},k))
	}
\end{equation*}

Noted that this is a form of SEQ-COND so expect a lot worse than COND (log linearly).

Similarly, we observed iid stream data $Y^t=\{Y_1,Y_2,\dots,Y_t\}$, where $Y_i=(Y_{i,a},Y_{i,b})^\intercal$ are the counts of ones in group $a, b$.
Numbers of ones in both group is denoted by $n_{i,1}=Y_{a,i}+Y_{b,i}$.

\subsubsection*{Conformal Prediction vs. Bayesian Methods}
\begin{itemize}
    \item Bayesian posterior probabilities give number in $(0,1)$ to 3 labels that sums to 1.
    \item A key difference is that conformal scores represent \emph{upper bounds} on probability and don't need to sum to one, which contrasts with standard probability theory.
    \item e.g., the conformal on the cat or the dogs can be both $20\%$, sort of the upper bound, 
    	while the human is 80\%, then it does not sum up to 1.
    \item Bayesian can be overconfident because of a prior, even a non-informative one, introduces some strong assumptions.
    \item Is there a real `FLAT' prior that provides no information?
    \item Conformal prediction can be seen as a ``worst-case'' approach, whereas Bayesian methods are ``best-case.'' Conformal may sometimes be too pessimistic.
\end{itemize}

\subsubsection*{Distinction Between Research Areas}
\begin{itemize}
    \item Sebastian is on \emph{model evaluation} and \emph{calibration}, which is more aligned with Bayesian statistics. His goal is to find the best overall predictor or model from a set of candidates, e.g., which weather forecasters is the best among 100.
    \item But conformal prediction focuses on the \emph{individual prediction} at each step, providing a set of possible outcomes with a certain confidence level (e.g., 95\%), without trying to determine the best underlying model.
    \item I am still not sure about how one formulate the conformal prediction:
    	Is it a sup over the training space?
\end{itemize}

\subsubsection*{Challenges with E-variables}
\begin{itemize}
    \item A major practical issue with sequential e-variables is that the cumulative product can be drastically reduced if multiplied by a single small e-value. This causes significant fluctuations.
    \item A simple ``safeguard'' is proposed: mixing the e-variable with a constant, such as using $0.5 + 0.5 \times E$. This ensures the multiplier is always at least 0.5, preventing a total loss of evidence.
    \item This is caused \emph{damping} and it's very popular in fin math where going to infinity
    is often not realistic and too theoretical!
    \item Using \emph{R\'enyi e-variables} is suggested as a potentially more sophisticated solution to this volatility problem.
\end{itemize}
